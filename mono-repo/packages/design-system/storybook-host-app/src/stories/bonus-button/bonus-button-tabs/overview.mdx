The _bonus button_ it is a button with `ds-bonus-button` selector, and it works in a way that you can put your text inside
and in a `[slot=value]` the value part and it will project the content for you.

<Canvas of={ButtonStories.Default} />

## Usage

Import `DsBonusButton` in your component, and apply `ds-bonus-button` in your buttons or anchor elements.

```ts

@Component({
    imports: [DsBonusButton], // ðŸ‘ˆ add to component imports
```
