The _icon button_ it is a button with `ds-icon-btn-icon` selector, and it works in a way that you can put your icon
and it will project it inside the button.

<Canvas of={ButtonStories.Default} />

## Usage

Import `DsIconButton` in your component, and apply `ds-icon-button` in your buttons or anchor elements.

```ts

@Component({
    imports: [DsIconButton], // ðŸ‘ˆ add to component imports
```
