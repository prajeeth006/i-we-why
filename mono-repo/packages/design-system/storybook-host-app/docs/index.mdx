# Welcome to Dice 

The global Design System of Entain and BetMGM, called Dice, is a reusable component library that helps contributors build UIs faster. The goal is to make building durable UIs more productive and satisfying.

<ul>
  <li><a href="https://jira.corp.entaingroup.com/secure/Dashboard.jspa?selectPageId=18515" target="_blank">Jira Dashboard</a></li>

</ul>

We show the current state of readiness for each component:

![Component Status Badges](docs-assets/component-status-badge.png "Component Status Badges")

* **Jira**: Link to the Jira Epic
* **Draft**: In development, it is not yet finished and may go through additional changes and testing
* **UX Review** The component is ready, but it requires sign of by UX
* **Experimental**: Can be integrated. UX updates and code change are likely to happen
* **Integration Ready**: The component can be picked by the teams and is ready for integration. UX updates and code changes can still happen
* **Stable**: The component is in use on production
* **A11y**: Accessibility is implemented and available in a component
